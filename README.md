# Dermatology Skin Bias Research

Bias-Aware Synthetic Augmentation for Multi-Condition 
Dermatological AI with Focus on Indian/South Asian Dark Skin Tones

## Datasets
- DermaCon-IN (IIT Hyderabad)
- DDI Stanford
- Fitzpatrick17k
- PASSION

## Pipeline
1. Data curation and filtering (FST V-VI)
2. LoRA fine-tuning of Stable Diffusion 2.0
3. Synthetic dark skin image generation
4. EfficientNet-B0 classifier training
5. Fairness evaluation
